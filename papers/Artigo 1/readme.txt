# **Artigo 1: Machine learning for predicting liver and/or lung metastasis in colorectal cancer**

Esta pasta contém os arquivos relacionados à reprodução e validação do artigo **"Machine learning for predicting liver and/or lung metastasis in colorectal cancer: A retrospective study based on the SEER database"**. Os conteúdos estão organizados em subdiretórios e arquivos para facilitar a navegação e o acompanhamento do processo de replicação.

---

## **Estrutura do Diretório**

### **1. V1**
Este subdiretório contém a primeira versão do código para a reprodução dos experimentos descritos no artigo. Inclui:
- Implementação de modelos de aprendizado de máquina.


### **2. datasets**
Neste subdiretório encontram-se os conjuntos de dados utilizados no estudo, incluindo:
- Análises exploratórias iniciais;
- Análise estatisticas da base de dados;
- **Base de Dados**: Base de dados gerada a partir da descrição do artigo.
- **Base de Dados Sintética**: Desenvolvida com base nos percentuais descritos no artigo.
- Arquivos de apoio para validação das etapas experimentais.


### **3. Artigo_01.pdf**
O arquivo PDF do artigo utilizado como referência para esta etapa do projeto. Ele serve como guia para a reprodução e validação dos experimentos.

### **4. readme.txt**
Um arquivo de texto contendo informações gerais sobre a organização e propósito desta pasta. Serve como um resumo rápido para os usuários do repositório.

---

## **Propósito**
O objetivo desta pasta é documentar, organizar e executar os experimentos baseados no artigo. Ela também permite:
- Reproduzir os resultados descritos no estudo original.
- Aplicar técnicas de aprendizado de máquina para validar os métodos no contexto médico.
- Servir como base para experimentações futuras no projeto.
